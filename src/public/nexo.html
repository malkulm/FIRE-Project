Environment Setup

To configure Nexo integration, add these environment variables to your .env file:

# Nexo API Configuration
NEXO_API_KEY=your_nexo_api_key_here
NEXO_API_BASE_URL=https://pg.nexo.com/api/v1
NEXO_WEBHOOK_SECRET=your_webhook_secret_here

API Testing

Test Nexo endpoints with curl:

# Check connection status
curl http://localhost:3000/api/nexo/status

# Get supported assets (requires API key)
curl -H "x-api-key: YOUR_NEXO_API_KEY" http://localhost:3000/api/nexo/assets

# Create deposit address
curl -X POST http://localhost:3000/api/nexo/addresses \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_NEXO_API_KEY" \
  -d '{"referenceId": "test-123", "asset": "BTC"}'

# View configuration
curl http://localhost:3000/api/nexo/config

Webhook Configuration

Configure your Nexo Payment Gateway dashboard to send webhooks to:

POST http://your-domain.com:3000/api/nexo/webhook

Supported webhook events:

  • PG_DEPOSIT - Cryptocurrency deposit received
